Working with Documents¶

You can work with documents associated with a feature or layer using the Documents view.

Contents:

  • Configure documents
  • Attaching a document
  • Linking a document
  • Setting a hotlink

Please note you must configure documents in Data ‣ Resource Properties to enable this functionality.

To list documents for a layer¶

  1. Open the Document view
  2. Select a layer in the Layers view
  3. The list of documents associated with this layer will be shown in a folder a folder

To list documents for a feature¶

  1. Open the Document view
  2. Select a feature using the Feature Selection tool
  3. The list of documents associated with this feature will be shown in the view in one folder, (followed by a separate folder for the layer documents).

Tips:

  • Ensure you are using the Feature Selection tool, the bounding box selection tool does not work with the document view at this time.
  • Configure the Information page to list your features by name (rather than ID).
