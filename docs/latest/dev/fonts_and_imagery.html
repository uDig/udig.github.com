Core Icon Concepts¶

UDIG constructs with well defined representations:

udig logo project file
feature grid
datastore grid coverge exchange
web feature server web map server
feature file grid file
map file pixel
repository web registry service
server database
project project missing
map empty map
map layer map layer empty
map folder map folder missing
paper page template

This is designed to intergrate well with the common Eclipse representations, and play with with the range of overlays etc.
