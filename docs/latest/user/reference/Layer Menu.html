Layer Menu¶

The Layer Menu is used to control the content displayed in the Map Editor. Many of the actions available in the Layer menu apply to the selected Layer as shown in the Layer view.

Add¶

Opens the Add Data wizard to add additional layers to the current map.

Create¶

Opens the Create Layer Dialog to create custom layers for the current map.

The dialog provides the ability to specify attribute fields for the layer. Restrictions related to the allowed length of the attribute value and whether null is allowed can also be specified
