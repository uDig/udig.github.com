Map Decoration page¶

The Map Decoration page lets you add decorations to your Map Editor.

After a map decoration has been added to your map it may be customized using the Style Editor dialog.

Several options are available:

  • Grid Decoration
  • JGrass Raster Legend
  • JGrass Vector Legend
  • Legend Decoration
  • North Arrow Decoration
  • Scalebar Decoration
  • Map Scale Decoration
  • The Processing Region
  • Graticule Decoration
