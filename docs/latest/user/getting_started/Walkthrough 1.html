Walkthrough 1¶

Walkthough the installation and use of the uDig application; covers the use of shapefile, Web Map Server, PostGIS and Web Feature Server.

  • uDig Walkthrough 1

This walk through makes use of the following downloads:

  • http://udig.refractions.net/download/ (latest release)
  • data_download (v1.3) (Thanks to the Natural Earth project)

The following web servies are used:

Web Map Server Web Feature Server
DM Solutions WMS DM Solutions WFS
OpenGeo Demo WMS 1.1.1 / WMS 1.3.0 OpenGeo Demo WFS 1.0.0 / WFS 1.1.0
Atlas of Canada WMS  

Tips:

  • The online help includes platform specific instructions for Running uDig. This page covers proxy settings.
  • Additional web services are listed in the web catalog
