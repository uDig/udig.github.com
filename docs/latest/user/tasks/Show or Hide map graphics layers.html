Show or Hide map graphics layers¶

Hiding map graphics layers¶

  1. Click on the Hide map graphics button on the view’s toolbar
  2. The view will hide all the map graphics layers from the list

Showing map graphics layers¶

  1. Click on the Show map graphics button on the view’s toolbar
  2. The view will show all the map graphics layers from the list
