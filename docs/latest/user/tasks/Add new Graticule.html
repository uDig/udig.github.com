Add new Graticule¶

  1. Select any map
  2. Left click Layer ‣ Add…
  3. Select Map Graticule
  4. Left click Next
  5. Left click Graticule
  6. Left click Finish

You should now see something similar to the example below

Note

Graticule only support CRS with unit “meter” on all axis.
