Configure documents¶

Configuration is required to enable resource documents, feature documents or feature hotlinks.

Support for documents is dependent on the resource selected and may not always be appropriate.

Configure resource documents¶

To enable resource documents (currently supported for shapefiles):

  1. Use Catalog view to select the resource (or select a layer from the Layers view).
  2. Select Data ‣ Resource Properties from the menu bar to open the Property Dialog.
  3. Navigate open Resource and select the Document page.
  4. From the Document page:
    • Check Enable support for resource documents

Configure feature documents¶

To enable feature documents (currently supported for shapefiles):

  1. Use Catalog view to select the resource (or select a layer from the Layers view).
  2. Select Data ‣ Resource Properties from the menu bar to open the Property Dialog.
  3. Navigate open Resource and select the Document page.
  4. From the Document page:
    • Check Enable support for feature documents

Configure feature hotlinks¶

To enable feature hotlinks:

  1. Use Catalog view to select the resource (or select a layer from the Layers view).
  2. Select Data ‣ Resource Properties from the menu bar to open the Property Dialog.
  3. Navigate open Resource and select the Document page.
  4. From the Document page:
    • Check Enable support for hotlink on marked attributes

To set an attribute as a file hotlink¶

  1. From the Document page
  2. Click Add… to open the Add Hotlink Descriptor dialog
  3. Select File as the Type and select the Attribute containing the reference information
  4. Provide the appropriate Label, and optionally Description
  5. Click OK

To set an attribute as a web hotlink¶

  1. From the Document page
  2. Click Add… to open the Add Hotlink Descriptor dialog
  3. Select Web as the Type and select the Attribute containing the reference information
  4. Provide the appropriate Label, and optionally Description
  5. Click OK

To set an attribute as an action hotlink¶

  1. From the Document page
  2. Click Add… to open the Add Hotlink Descriptor dialog
  3. Select Action as the Type and select the Attribute containing the reference information
  4. Provide the appropriate Label, and optionally Description
  5. Provide the Action definition, using the marker ‘{0}’ to indicate where the attribute value is used.
  6. Click OK

Example Actions:

  • Google search: https://www.google.com.au/search?q={0}
  • Yahoo search: http://au.search.yahoo.com/search?p={0}
  • File reference: C:\\Reports\\{0}.pdf
  • Open in application: notepad {0}

To edit an existing hotlink definition¶

  1. From the Document page
  2. Select hotlink from the list
  3. Click Edit… to open the Edit Hotlink Descriptor dialog
  4. Select Type
  5. Select the Attribute containing the reference information
  6. Provide the appropriate Label
  7. (Optional) Provide the appropriate Description
  8. (For Action types) Provide the Action definition
  9. Click OK

To remove an existing hotlink definition¶

  1. From the Document page
  2. Select hotlink from the list
  3. Click Remove

Sample dataset:

  • Try out australia.shp in data_1_4.zip of the Quickstart sample data.
