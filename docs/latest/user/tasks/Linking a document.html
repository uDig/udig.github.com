Linking a document¶

To link a file document to a layer or feature:¶

  1. Load documents related to layer or feature
  2. Select the layer or feature’s folder or a document inside the folder
  3. Click Link… to open the Add Link dialog
  4. Set File as the Type
  5. Provide the file to be linked by doing one of the following:
  • Manually set File with the file’s path
  • Select an existing file
    • Click Browse… to open the file selection dialog
    • Select a file from the dialog
    • Click Open to set File with the selected file’s path
  • Create a new file from template
    • Click New… to open the select template dialog
    • Select a template from the list
    • Click OK to open the save file dialog
    • Select directory and enter filename
    • Click Save to set File with the new file’s path
  1. Set the Label with appropriate label
  2. (Optional) Set the Description with appropriate description
  3. (Optional) Click Open to verify that set file is valid and existing
  4. Click Link to save link

To link a web document to a layer or feature:¶

  1. Load documents related to layer or feature
  2. Select the layer or feature’s folder or a document inside the folder
  3. Click Link… to open the Add Link dialog
  4. Set Web as the Type
  5. Set URL with the website URL to be linked
  6. Set the Label with appropriate label
  7. (Optional) Set the Description with appropriate description
  8. (Optional) Click Open to verify that set URL is valid
  9. Click Link to save link

Note: Certain resource types only allow linking certain document types. For example, shapefiles only allow linking of web documents.

Sample dataset:

  • Try out australia.shp in data_1_4.zip of the Quickstart sample data.
