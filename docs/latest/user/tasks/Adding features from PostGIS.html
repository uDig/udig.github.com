Adding features from PostGIS¶

  1. Select Layer ‣ Add … from the menu bar to open the Add Data wizard
  2. Select PostGIS from the list and press Next.
  3. Fill in the PostGIS page:
    • Enter your PostGIS host and authentication parameters
    • Select the database connect and optionally a schema
  4. Press Next
  5. Select they layer(s) to be added
  6. Press Finish

Layers will be added to the current map (created if needed). The PostGIS server defined will be added to the Catalog view
