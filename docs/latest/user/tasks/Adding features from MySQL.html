Adding features from MySQL¶

  1. Select Layer ‣ Add … from the menu bar to open the Add Data wizard

  2. Select MySQL Host from the list and press Next.

  3. Fill in the MySQL page:

    • Enter your MySQL host and authentication parameters
    • Select the database connect and optionally a schema

  4. Press Next

  5. Select the layer(s) to be added

  6. Press Finish

Layers will be added to the current map (created if needed). The MySQL server defined will be added to the Catalog view
