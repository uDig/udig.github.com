Changing a layer’s z-order¶

Moving a layer to the front¶

  1. Select the layer to be moved
  2. Click on the Move to front button on the view’s toolbar
  3. Your layer will be moved to the front

 Moving a layer forward¶

  1. Select the layer to be moved
  2. Click on the Move forward button on the view’s toolbar
  3. Your layer will be moved forward ahead of the next layer

Moving a layer backward¶

  1. Select the layer to be moved
  2. Click on the Move backward button on the view’s toolbar
  3. Your layer will be moved backward behind the next layer

Moving a layer to the back¶

  1. Select the layer to be moved
  2. Click on the Move to back button on the view’s toolbar
  3. Your layer will be moved to the back

*These changes will not be visible in the Legend View but will be seen on the layer’s ordering in the Layers View and on the map editor.
