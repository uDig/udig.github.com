Setting a hotlink¶

To list hotlinks related to a feature:¶

  1. Open the Document view
  2. Select a feature using the Feature Selection tool
  3. The select feature is presented as a folder, containing any associated documents including hotlinks.

To set a file hotlink:¶

  1. Load documents related to feature
  2. Select the hotlink from the list
  3. Click Edit... to open the Edit File Hotlink dialog
  4. Provide the file to be linked by doing one of the following:

  • Manually set Value with the file’s path

  • Select an existing file
    • Click Browse... to open the file selection dialog
    • Select a file from the dialog
    • Click Open to set Value with the selected file’s path
  • Create a new file from template
    • Click New... to open the select template dialog
    • Select a template from the list
    • Click OK to open the save file dialog
    • Select directory and enter filename
    • Click Save to set Value with the new file’s path
  1. (Optional) Click Open to verify that set file is valid and existing
  2. Click OK to set the hotlink value
  3. Click Commit to write this change out as the hotlink is stored in the assigned attribute

To set a web hotlink:¶

  1. Load documents related to feature
  2. Select the hotlink from the list
  3. Click Edit... to open the Edit File Hotlink dialog
  4. Set Value with the website URL to be linked
  5. (Optional) Click Open to verify that set URL is valid
  6. Click OK to set the hotlink value
  7. Click Commit to write this change out as the hotlink is stored in the assigned attribute

To set an action hotlink:¶

  1. Load documents related to feature
  2. Select the hotlink from the list
  3. Click Edit... to open the Edit File Hotlink dialog
  4. Set Value with the appropriate attribute value
  5. (Optional) Select Action and click Go to verify the value with set actions
  6. Click OK to set the hotlink value
  7. Click Commit to write this change out as the hotlink is stored in the assigned attribute

Sample dataset:

  • Try out australia.shp in data_1_4.zip of the Quickstart sample data.
