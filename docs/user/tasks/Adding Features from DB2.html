Adding Features from DB2¶

There are a number of options for adding a DB2 layer to your map.

  1. Open the Add Data dialog with one of the following:
  • Press the New button on the toolbar
  • Select Add item from the Layer menu
  • Select New ‣ Layer from the File menu
  1. If you have not set your DB2 preference (Window ‣ Preferences...) then you will have the chance to do so now. See DB2 Preferences for instructions on how to do this
  2. Select DB2 from the Wizard and press Next
  3. Fill in the DB2 page
    • Enter your DB2 host, authentication parameters and database name
    • Press the Get schemas button to populate the schema drop-down list
    • Select a schema from the drop-down list
  4. Press Next
  5. From the Resource Selection page select the layer(s) that will compose the new map
  6. Press Finish

The selected DB2 tables will be added to the Catalog and a new map containing selected Layers will be displayed.
