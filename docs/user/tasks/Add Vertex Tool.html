Add Vertex Tool¶

Add Vertex Tool – Cannot resolve external resource into attachment.¶

The Add Vertex Tool can select geometries and add vertices to those geometries. The workflow is the same as adding vertices using the Edit Geometry Tool.

  1. Click a feature to select it. Tip if features overlap clicking the same location multiple times will cycle through the features at that point.
  2. Click on an edge to add a vertex.
  3. Double click to accept change.
