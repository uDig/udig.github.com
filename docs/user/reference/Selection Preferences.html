Selection Preferences¶

Available Preferences¶

  • Selection radius - controls the snap radius (in pixel) during single feature selection like command. On mouse click on the map all features found within this radius will be appear in an popup (using the default feature attribute mentioned in Tool Preferences) for the user to select from (see figure below)

  • Area of Interest (AOI) Selection - Checkbox to enable change to next available AOI layer on selection. Default is enabled.
